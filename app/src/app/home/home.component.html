Welcome to the MementoCoding Project!

This project is a comprehensive School Management System built using the latest technologies - .NET 6 and Angular 15. It is designed to streamline and enhance the management of schools and the information associated with students, providing an efficient and user-friendly solution for educators and administrators.

Key Features

  • Guest User Access: Guests can use search engine features to access school information.
  • Admin Role: Administrators can modify and manage data as needed.
  • School Information: Easily manage schools informations with students linked to it
  • Student Information: Easily manage student profiles, including personal details
  • Search Engine: Easily search for any fields or info in the DB

Upcoming Key Features (Planned)

  • Class and Course Management: Plan to organize classes and courses, assign teachers, and schedule lessons.
  • Grading and Report Cards: In development to record grades, generate report cards, and keep parents informed of their child's progress.
  • User Authentication: Planned for secure user authentication and authorization for teachers, administrators, and students.
  • Communication Hub: A forthcoming feature to facilitate communication between teachers, students, and parents.
  • Data Analytics: On the roadmap to utilize data analytics to make informed decisions and enhance school operations.

Technology Stack

Our project leverages the power of .NET 6 for the backend, providing a robust and scalable foundation for data management, business logic, and API development. On the frontend, Angular 15 offers a modern and responsive user interface for accessing and interacting with the system.
